# clawdbotv1

ClawdBot v1 — personal AI assistant bot with tool use.

## Stack

TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/clawdbotv1.git
cd clawdbotv1
bun install
bun run dev   # or: bun run start
```
